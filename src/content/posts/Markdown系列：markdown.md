## 一级标题 (h1)

段落之间由空行分隔。

这是第二段。支持 *斜体*、**粗体** 和 `等宽字体`。项目列表如下：

  - 这一项
  - 那一项
  - 另一项

请注意，除去星号外，实际的文本内容是从第 4 列开始的。

> 块引用（Block quotes）
> 是这样写的。
> 如果你愿意，它们可以跨越多个段落。

使用三个连字符表示破折号 ---。使用两个连字符表示范围（例如，“都在第 12--14 章”）。三个点 ... 将被转换为省略号。支持 Unicode 字符。 ☺

## 二级标题 (h2)

这是一个数字列表：

1.  第一项
2.  第二项
3.  第三项

再次注意，实际文本是从左侧起第 4 个字符开始的。这是一个代码示例：

```plaintext
1 # 让我再重复一遍...
2 for i in 1 .. 10 { do-something(i) }
```

正如你可能猜到的，它缩进了 4 个空格。顺便说一下，如果你愿意，可以使用定界块（delimited blocks）代替缩进：

```plaintext
define foobar() {
    print "欢迎来到风味国度!";
}
```

（这使得复制和粘贴更容易）。你可以选择为定界块标记语言，以便 Pandoc 进行语法高亮：

```python
import time
# 快，数到十！
for i in range(10):
    # （但不要*太*快）
    time.sleep(0.5)
    print i
```

## 三级标题 (h3)

现在是一个嵌套列表：

1.  首先，准备这些配方：
      - 胡萝卜
      - 芹菜
      - 小扁豆
2.  烧开水。
3.  把所有东西倒进锅里，并遵循以下算法：
    ```plaintext
    1 找木勺
    2 揭开锅盖
    3 搅拌
    4 盖上锅盖
    5 将木勺颤巍巍地平衡在锅柄上
    6 等待 10 分钟
    7 回到第一步（或者如果做好了就关火）
    ```

不要碰到木勺，否则它会掉下来。

再次注意，文本始终在 4 空格缩进处对齐（包括上面延续第 3 项的最后一行）。

这是一个指向 [一个网站](http://foo.bar/) 的链接，指向 [本地文档](https://www.google.com/search?q=http://localhost:4321/posts/markdown/local-doc.html) 的链接，以及指向 [当前文档中某个章节标题](https://www.google.com/search?q=http://localhost:4321/posts/markdown/%23an-h2-header) 的链接。这是一个脚注 [^1]。

表格可以长这样：

| 尺码 | 材质 | 颜色 |
| :--- | :--- | :--- |
| 9 | 皮革 | 棕色 |
| 10 | 麻帆布 | 原色 |
| 11 | 玻璃 | 透明 |

表：鞋子，它们的尺寸以及材质

（上方是表格的标题。）Pandoc 也支持多行表格：

| 关键词 | 文本 |
| :--- | :--- |
| 红色 | 日落、苹果以及其他红色或偏红的事物。 |
| 绿色 | 叶子、草、青蛙以及其他不容易生存的事物。 |

下面是一个水平分割线。

-----

这是一个定义列表：

**苹果**
: 适合制作苹果酱。

**橙子**
: 柑橘属！

**西红柿**
: tomatoe 这个词里没有“e”。

同样，文本缩进 4 个空格。（在每对术语/定义之间留一行空行可以将它们分得更开。）

这是一个“行块”：
| 第一行
| 第二行
| 第三行

行内数学公式如下输入：$\omega = d\phi / dt$。展示型数学公式应独占一行并置于双美元符号中：
$$I = \int \rho R^2 dV$$
$$\pi = 3.1415926535 \dots$$

请注意，你可以对任何希望按原样显示的标点符号进行反斜杠转义，例如：\`foo\`、\*bar\* 等。
